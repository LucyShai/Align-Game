# Alien Invasion Game

A classic arcade-style shooter built with Python and Pygame.

## Features

- **Progressive Difficulty**: 10 levels of increasing speed and alien fleet size.
- **Triangular Formations**: Dynamic fleet spawns in pyramid/triangular shapes.
- **Visual UI**: Life indicators and level tracking on-screen.
- **Starfield Background**: Dynamic space visuals.

## How to Run

1. Install Pygame:

   ```bash
   pip install pygame
   ```

2. Run the game:

   ```bash
   python "Part 3/alien_invasion.py"
   ```

## Controls

- **Arrow Keys**: Move the ship.
- **Spacebar**: Fire bullets.
- **Q**: Quit.
